# One-Pager Web Application

## Purpose

This directory contains the web application for the One-Pager Assistant example project.

## Contents

- `index.html` - Main HTML file
- `js/` - JavaScript modules
- `css/` - Custom styles
- `data/` - Prompt templates

## Running Locally

```bash
# From this directory
python3 -m http.server 8000

# Open browser
open http://localhost:8000
```

## Features

- 3-phase workflow (Draft → Review → Finalize)
- IndexedDB storage
- Dark mode
- Export to Markdown
- Responsive design
